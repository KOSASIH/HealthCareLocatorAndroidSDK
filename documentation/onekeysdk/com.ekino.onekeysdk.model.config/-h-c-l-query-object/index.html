HCLQueryObject

data class HCLQueryObject

HCLQueryObject provides the parameters for fetching data from the service without UI.

Parameters

id - is the id of activity or individual.

first - the size of list data which is returning from web service.

offset - the page of the results.

criteria - is searching the results from user's input.

locale - is the language for results.

userId - is the user identity.

specialities - is the list of speciality id.
